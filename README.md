# COM206-midterm project by Taha Adem
Welcome to the COM206-miterm project repository.
This repository is designed to publish my project for my "COM206 Visual Programming" midterm project.

# Project Overview
This project is a first step into using flutter for me to create a basic application using dart and to learn about Graphical User Interfaces (GUI) as both philosophy and a programming practice.

This project was made for android phones in consideration but could work on other devices.


# Task Flow
The application uses stacks to move between pages. first page gets pushed in and then the second page. and pops when returning back to first page.
