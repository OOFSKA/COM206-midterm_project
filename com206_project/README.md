# Application Basics

A flutter project for my midterm.

## Getting Started

To use this application you need to run the "main.dart" on your own machine after installing the files,
the application is made up of two pages, a Home Page and a Buttons page.
The home page consists of a button leading to the other page that has few buttons to play around with even though they all are almost unfunctional.

This project was mainly made to learn about using flutter and dart in my way to visual programming and has no real benefit to anything else except my grades.
